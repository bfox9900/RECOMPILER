These files are source files and program files for the TI-99 computer.
The source files are in DV80 format and can be edited with EDIT1 and the
Editor/Assembler Cartridge installed.  

The program files are METAFORTH METAFORTI METAFORTJ 
They are a Meta compiler program called METAFORTH.
Load METAFORTH with the Editor/Assembler cartridge, menu option 5
